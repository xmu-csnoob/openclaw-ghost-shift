# Ghost Shift UI 重构任务 v2

## 背景
Ghost Shift 是一个对外发布的公开遥测产品，需要：
1. 完整中文化
2. 简化 UI，让像素办公室成为主角

## 核心问题

### 问题1：中文化不完整
- 当前仍有大量英文残留
- 需要系统性检查所有用户可见文字

### 问题2：UI 太臃肿
当前页面堆砌了太多功能面板：
- 体验控制面板（Guide、Settings）
- 产品仪表盘（雷达图、散点图、柱状图）
- 分享面板（生成预览、复制链接、下载PNG等）
- 案例研究层
- 统计侧边栏

**这些都应该简化或隐藏**

## 重构方案

### 1. 页面布局简化
```
┌─────────────────────────────────────────┐
│  导航栏（品牌 + 路由链接）                │
├─────────────────────────────────────────┤
│                                         │
│         像素办公室（全屏主角）            │
│                                         │
│         [少量悬浮控制按钮]               │
│                                         │
├─────────────────────────────────────────┤
│  底部状态栏（实时状态 + 基础统计）        │
└─────────────────────────────────────────┘
```

### 2. 功能降维
将复杂面板改为按钮触发的子窗口/弹窗：

| 原功能 | 改为 |
|--------|------|
| 分享面板（完整卡片生成器） | 一个"分享"按钮 → 点击弹出 Modal |
| 体验设置（主题、密度等） | 一个"设置"图标 → 点击弹出 Modal |
| 案例研究层 | 一个"？"图标 → 点击弹出 Modal |
| 产品仪表盘（雷达图等） | 移除或折叠到"统计"按钮 |
| 回放控制条 | 保留，但简化为底部极简条 |

### 3. 保留的核心元素
- 顶部导航栏：品牌名 + 路由链接（实时/回放/嵌入/文档/关于）
- 像素办公室画布：全屏显示，这是产品核心
- 底部状态栏：实时状态、Agent数量、更新时间
- 回放控制：简化版，只保留播放/暂停 + 时间轴

### 4. 隐藏到弹窗的功能
- 分享（生成社交卡片）
- 设置（主题、密度）
- 帮助/案例说明
- 详细统计

## 技术要求

### 中文化
- 扫描所有 .tsx 文件，找出所有英文文案
- 使用 `src/content/i18n.ts` 统一管理
- 措辞专业，面向外部用户

### UI 重构
- 创建 Modal 组件用于弹窗功能
- 简化 GhostShiftSurface.tsx 的渲染结构
- 移除或折叠冗余面板
- 让办公室画布占据主要空间
